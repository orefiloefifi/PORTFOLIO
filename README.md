# 🚀 Overview

**LGK Portfolio** is a professional portfolio website developed to showcase my work as a **Frontend Developer, Flutter Developer, and UI/UX Designer**. The platform highlights featured projects, technical skills, experience, and creative solutions through an elegant, responsive, and interactive interface.

Built using modern web technologies, the portfolio emphasizes clean design, performance, and accessibility while providing visitors with an engaging way to explore my work and professional journey.

